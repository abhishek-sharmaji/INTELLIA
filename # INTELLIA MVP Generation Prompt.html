# INTELLIA MVP Generation Prompt

You are an expert Full-Stack Software Engineer and UI/UX Designer.

Build a **production-ready MVP** for a fintech web application called **INTELLIA – AI Financial Dashboard**.

## Goal

Create a clean, modern, working MVP suitable for a hackathon demo. Focus on quality over quantity.

## Tech Stack

* React + Vite + TypeScript
* Tailwind CSS
* Node.js + Express
* Firebase Authentication
* Firebase Firestore
* OpenAI API
* Chart.js
* Lucide React

## Pages

1. Landing Page
2. Login
3. Sign Up
4. Dashboard
5. Settings

## Features

* User authentication with Firebase
* Protected dashboard
* Add, edit, delete transactions
* Financial summary cards (Income, Expense, Savings, Balance)
* Expense chart using Chart.js
* AI Spending Analysis using OpenAI API
* Simple AI chatbot for financial questions
* Dark/Light mode
* Responsive design
* Toast notifications
* Loading states
* Form validation

## Dashboard

Display:

* Total Balance
* Monthly Income
* Monthly Expenses
* Savings
* Recent Transactions
* Expense Breakdown Chart
* AI Insights Panel
* Budget Progress Bar

## AI Features

* Analyze spending habits
* Suggest ways to save money
* Recommend a monthly budget
* Answer simple finance-related questions

## Database

Firestore collections:

* users
* transactions

Each transaction contains:

* title
* amount
* category
* date
* type (income/expense)

src/

server/

## UI Design

Create a modern interface inspired by Stripe and Linear:

* Glassmorphism cards
* Rounded corners
* Soft shadows
* Gradient accents
* Mobile responsive
* Accessible components
* Smooth hover effects
* Professional typography
* Consistent spacing

## Code Quality

* TypeScript throughout
* Reusable components
* Custom hooks
* Context API for authentication and theme
* Proper error handling
* Clean architecture
* No duplicated code

## Deliverables

Generate:

* Complete folder structure
* All source code
* Backend APIs
* Firebase configuration
* README with setup instructions

The application should run after installing dependencies, configuring Firebase and the OpenAI API key, and executing the frontend and backend servers.
